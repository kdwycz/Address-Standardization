地址行政区化和对比小工具
=====================
介绍
----
在Debian Linux系统上使用Python2.7编写的小工具，用来把确定但杂乱的地址整理成 省-市-区（县）-街道 的形式。并根据 @曲彦文 老师的要求，把这些地址和其他地址做比对，是否在同一省/市/区。

其实这个项目毫无意义，做自己的Github练手用吧


依赖库
----
- httplib2
- json
- codecs


使用方法
----
把表格的数据复制到input文件夹下对应的txt文件。

运行根目录下 start.py 文件

注意事项
----
- 最好在linux系统下复制excel到文本。
